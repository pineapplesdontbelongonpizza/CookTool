# CookTool

## Overview

Smart Kitchen is a machine learning based restaurant demand forecasting system that helps restaurants decide how much food to prepare daily.

The goal is to reduce food wastage while ensuring enough inventory is available for customers.

## Problem

Restaurants often struggle with:
- Over-preparation leading to food waste
- Under-preparation causing missed sales

## Solution

The application predicts expected food demand based on:

- Dish type
- Day of week
- Temperature
- Holiday information
- Previous sales


## Features

- Food demand prediction  
- Restaurant dashboard  
- Sales trend visualization  
- Waste estimation  
- Data-driven preparation recommendations  


## Tech Stack

- Python
- Pandas
- Scikit-learn
- Random Forest Regression
- Streamlit
